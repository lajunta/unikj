输入与输出

基本数据类型

  • 字符 (char): 用单引号包围的单个字符，例如 'c', '1'。
  • 整数 (int): 包括负整数、零和正整数，例如 -10, 0, 100。
  • 浮点数 (float): 带小数点的数字，例如 3.14, -0.001。通常精度为6-7位有效数字。
  • 双精度浮点数 (double): 精度和范围比 float 更高的小数，例如 3.141592653589793。

其它数据类型

  • 空类型 (void): 表示无类型，常用于函数返回值或指针。
  • 数组类型: 存储一组相同类型数据的集合。
  • 结构体类型: 用户自定义的数据类型，可以包含不同类型的数据。
  • 指针类型: 存储变量内存地址的数据类型。

变量

变量可以理解为一个存储数据的容器（盒子），给这个容器一个名字，就是变量名。

容器的名称可以自己定义，容器中存储的内容也可以随时改变。

定义变量

在使用变量之前，需要先声明它的类型和名称。可以在声明时初始化变量。

// 声明并初始化整型变量
int age = 16;
int math = 85;
int chinese = 88;
int english = 99;

// 声明并初始化字符型变量
char isChinese = 'Y';
char isJapanese = 'N';

// 声明并初始化浮点型变量
float shengao = 1.80f; // 也可以写成 1.80
float height = 1.80f;
float weight = 65.8f;
float pi = 3.14159f;

格式化输出 printf

printf 是 C 语言中用于输出信息到屏幕的标准库函数，定义在 <stdio.h> 头文件中。它支持格式化输出字符串和变量的值。

基本语法

printf("格式字符串", 变量1, 变量2, ...);

常用格式符

  • %d：输出十进制整数
  • %f：输出浮点数
  • %c：输出单个字符
  • %s：输出字符串
  • %.2f: 输出浮点数，保留两位小数
  • %5d: 输出整数，至少占5个字符宽度，右对齐

示例

#include <stdio.h>

int main() {
    int age = 18;
    float height = 1.75f;
    char sex = 'M';
    char name[] = "张三";
    
    printf("我的姓名：%s\n", name);
    printf("我的年龄：%d\n", age);
    printf("我的身高：%.2f米\n", height);
    printf("我的性别：%c\n", sex);
    printf("打个招呼：%s\n", "Hello, How are you?");
    
    // 格式化输出示例
    int number = 42;
    printf("数字(默认)：%d\n", number);
    printf("数字(5位宽)：%5d\n", number);
    printf("数字(前导0)：%05d\n", number);
    
    float price = 123.456f;
    printf("价格(默认)：%f\n", price);
    printf("价格(保留2位小数)：%.2f\n", price);
    printf("价格(总宽10,保留2位小数)：%10.2f\n", price);
    
    return 0;
}

输出结果:

我的姓名：张三
我的年龄：18
我的身高：1.75米
我的性别：M
打个招呼：Hello, How are you?
数字(默认)：42
数字(5位宽)：   42
数字(前导0)：00042
价格(默认)：123.456001
价格(保留2位小数)：123.46
价格(总宽10,保留2位小数)：    123.46

定义字符串并输出

在 C 语言中，可以使用字符数组来定义字符串，并通过 printf 和 %s 格式符输出：

#include <stdio.h>

int main() {
    // 定义并初始化字符数组（字符串）
    char greeting[] = "你好，欢迎学习C语言！";
    char course[] = "C语言基础";
    
    printf("%s\n", greeting);
    printf("当前学习课程：%s\n", course);
    
    // 定义固定长度的字符数组
    char message[50] = "祝学习愉快！";
    printf("祝福语：%s\n", message);
    
    return 0;
}

输出结果:

你好，欢迎学习C语言！
当前学习课程：C语言基础
祝福语：祝学习愉快！

格式化输入 scanf

scanf 是 C 语言中用于从标准输入（通常是键盘）读取数据的标准库函数，定义在 <stdio.h> 头文件中。它可以读取不同格式的数据并存储到变量中。

基本语法

scanf("格式字符串", &变量1, &变量2, ...);

注意：对于基本数据类型（如 int, float, char）的变量，在变量名前需要加 &（取地址运算符），表示将输入的数据存储到该变量的内存地址中。对于字符数组（字符串），则不需要 &。

示例1：输入年龄和身高

#include <stdio.h>

int main() {
    int age;
    float height;
    
    printf("请输入你的年龄和身高（用空格分隔）：");
    scanf("%d %f", &age, &height);
    
    printf("你的年龄是：%d岁，身高是：%.2f米\n", age, height);
    
    return 0;
}

输入示例: 20 1.75 输出结果: 你的年龄是：20岁，身高是：1.75米

示例2：输入姓名、性别和成绩

#include <stdio.h>

int main() {
    char name[20];
    char gender;
    float score;
    
    printf("请输入你的姓名（不含空格）、性别（M/F）和成绩（用空格分隔）：");
    scanf("%s %c %f", name, &gender, &score);
    
    printf("学生信息：姓名=%s, 性别=%c, 成绩=%.1f\n", name, gender, score);
    
    return 0;
}

输入示例: 李四 M 95.5 输出结果: 学生信息：姓名=李四, 性别=M, 成绩=95.5

示例3：输入多个整数

#include <stdio.h>

int main() {
    int a, b, c;
    
    printf("请输入三个整数（用空格分隔）：");
    scanf("%d %d %d", &a, &b, &c);
    
    printf("你输入的三个数分别是：%d, %d, %d\n", a, b, c);
    printf("它们的和是：%d\n", a + b + c);
    
    return 0;
}

输入示例: 10 20 30 输出结果: 

你输入的三个数分别是：10, 20, 30
它们的和是：60
